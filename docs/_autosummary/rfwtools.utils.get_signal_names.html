rfwtools.utils.get_signal_names¶

rfwtools.utils.get_signal_names(cavities, waveforms)[source]¶

Creates a list of signal names by joining each combination of the two lists with _

Parameters

  • cavities (List[str]) – A list of strings that represent cavity numbers, e.g. ‘1’ or ‘7’. These are the cavities for which signals will be included.

  • waveforms (List[str]) – A list of waveform suffixes (e.g., “GMES” or “CRRP”) for the waveforms to be included in the output.

Return type

List[str]

Returns

The list containing all of the combinations of the supplied cavities and waveforms
